InternHub - Internship Management System
InternHub is a comprehensive web application designed to streamline the management of internship programs. It provides a user-friendly interface for administrators to oversee candidates, manage internship postings, and track application statuses. It allows candidates to 
see new opportunities (internships). 

==> Features:

-> Dashboard Overview: Get a quick glance at key metrics and recent activities.
-> Candidate Management: Add, view, search, filter, and manage candidate applications.
-> Internship Management: Post new internships, edit existing ones, and track applicant numbers.
-> User Authentication: Secure login and session management.
-> Responsive Design: Optimized for both desktop and mobile devices.
-> Notifications: Receive real-time updates on new applications and other important events.

Technologies Used
React.js: Frontend library for building user interfaces.
Tailwind CSS: A utility-first CSS framework for rapid styling.
Lucide React: A collection of beautiful open-source icons.
React Router DOM: For declarative routing in React applications.
Local Storage: For persisting user authentication and theme preferences.

Installation:
Follow these steps to get InternHub up and running on your local machine.
Prerequisites
Node.js (LTS version recommended)
npm or Yarn

Steps:
Clone the repository:
git clone <your-repository-url>
cd intern-hub

(Replace <your-repository-url> with the actual URL of your GitHub repository.)

Install dependencies:
npm install
# or
yarn install

Run the development server:
npm start
# or
yarn start


Usage
Login: Use test@example.com as the email and password123 as the password to log in.
Navigation: Use the sidebar to navigate between Dashboard, Candidates, Internships, and Profile pages.

Candidates Page:
Search for candidates by name or email.
Sort candidates by name or status.
Add new candidates using the "Add Candidate" button.
View candidate details in the table.

Internships Page:
Search for internships by title or department.
Filter internships by department or status.
Post new internships using the "Post New Internship" button.
